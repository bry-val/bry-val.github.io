Explanation

My first visualization is a scatter plot that demonstrates a positively correlated relationship between the low temperature variable and the dewpoint.

I tried many different variables in this scatter plot and found these two to demonstrate the best relationship. I am not sure how dew point is measured excatly, so I do not know if this is graphing redundant data. The way this data is set up makes me think that dew point is a calculation with temperature_low as one of its factors.

I chose the interval selector here because I like the way it works as an interactive plot. I think making my two visualizations an interactive dashboard that work in unison was effective for this because it allowed me to grant greater insight about the distribution of the different ends of the spectrum for low temperatures and dew points.

The histogram here demonstrates an identical color pattern but uses a an aggregate function that counts each occurance grouped by their “Season” value. I chose to include Unknown because I think it helps to keep our data interesting.

As the data are linked I will talk about the datasets mostly together.

For my scatter plot I used quantitative encoding for both the X and the Y, and for my histogram I used quantitative for the count aggregation and nominal for the seasons, as seasons is a collection of strings.

The interactivity here makes my dashboard interesting because you can modify the histogram with the interval selector. I really like this feature and would like to implement it with a more robust data set.

I think out of all the visualization platforms we have used, vega-lite has been my favorite. It appears very full featured and the altair integration makes data modification (which I did not do here, but may do in the future) incredibly easy.
